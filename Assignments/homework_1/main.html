Object Oriented Programming

Main Abstraction Encapsulation Inheritance Polymorphism


What is object oriented programming?

It is a way of programming which focuses on the use of objects. Object oriented programming focuses on identifying all of the objects that will be used, manipulating these objects, and understanding how they relate to eachother.

Pros?

This form of programming usually simplifies the thought process behind programming by breaking ideas up into smaller sections(objects).

Cons?

Some cons of object oriented programming include being inefficient and unintuitive once codebases become too large.
